# Morphology Study
A repos for USTH Digital Image Processing 2020 Group 1. It's quite obvious in the title.

## Morphological transform list
- Open, close

- Erode, dialate

- Gradient

- Hit or miss

- Skeleton

- Tophat, blackhat

## Requirements
Python 3
```
pip install -r requirements.txt
```
